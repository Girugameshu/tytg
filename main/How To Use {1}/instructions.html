Setting up tytg is really easy! First of all, you should install pip. Then, install the setuptools module: pip install setuptools Install dependancies: python setup.py install And you're done. In order to run the bot use: python tytg.py [ROOT] [TOKEN] Where the root is the starting point for the bot, and the token is given by botfather.
